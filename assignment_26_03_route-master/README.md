# Assignment 26/03/2026 - Route Master

Build Express routes for a bookstore.

## Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Use these routes:
   - `GET /api/books`
   - `GET /api/books/:id`
   - `GET /api/authors`
   - `GET /api/authors/:id`

## Summary

This assignment demonstrates Express routing by separating the bookstore routes into `books` and `authors` modules.

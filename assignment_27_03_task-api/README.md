# Assignment 27/03/2026 - Task API

Create CRUD APIs for a task manager.

## Run

```bash
npm install
npm start
```

## Endpoints

- `GET /api/tasks`
- `GET /api/tasks/:id`
- `POST /api/tasks`
- `PUT /api/tasks/:id`
- `DELETE /api/tasks/:id`

## Postman

Use Postman to send requests to the routes above and verify the CRUD behavior.
